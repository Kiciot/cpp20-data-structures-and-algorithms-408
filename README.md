# C++20 数据结构与算法实现

一个基于 **C++20** 的数据结构与常见算法实现项目，面向 **408 复试 / 机试准备** 以及希望系统掌握数据结构内容的学习者。

本项目以“**工程实现 + 知识点覆盖**”为目标，尽量将常见数据结构与算法拆分为独立的 `.cpp` 文件，便于查阅、复习、对照和练习，而不必在学习过程中到处零散查找资料。

## 项目定位

- 面向 408 复试、机试准备者
- 面向希望系统学习数据结构与算法实现的 C++ 学习者
- 使用 **C++20** 编写
- 强调“**按知识点拆分的工程实现**”，便于逐个理解与练习

## 项目特点

- 覆盖数据结构核心章节与常见算法
- 每个主题独立实现，便于单独编译与学习
- 适合作为：
  - 408 复试 / 机试代码参考
  - 数据结构课程复习资料
  - C++ 数据结构实现练习仓库
  - 面试前基础回顾材料
