# C++20 数据结构与算法实现

一个基于 **C++20** 的数据结构与常见算法实现项目，面向 **408 复试 / 机试准备**、数据结构课程学习，以及希望系统掌握数据结构内容的学习者。

本项目以“**工程实现 + 知识点覆盖 + 便于复习**”为目标，将常见数据结构与算法拆分为独立的 `.cpp` 文件，并按章节组织到不同目录中，便于查阅、复习、对照、单独编译与练习，而不必在学习过程中零散寻找代码模板。

---

## 项目定位

本项目主要服务于以下场景：

- **408 复试 / 机试准备**

- **C++ 数据结构工程实现入门**
- **常见数据结构与算法的系统性复习**

相较于偏理论化或偏刷题化的资料，本项目更强调：

- 用 **C++20** 完成典型数据结构与算法实现

## 快速开始

### 1. 克隆仓库

```bash
git clone git@github.com:Kiciot/cpp20-data-structures-and-algorithms-408.git
cd cpp20-data-structures-and-algorithms-408
```

如果你使用 HTTPS：

```bash
git clone https://github.com/Kiciot/cpp20-data-structures-and-algorithms-408.git
cd cpp20-data-structures-and-algorithms-408
```

---

### 2. 编译单个文件

由于当前项目采用 **单文件独立实现** 的方式，因此推荐按主题逐个编译。

例如编译 KMP：

```bash
g++ -std=c++20 -O2 "04-string/4.2.kmp算法.cpp" -o kmp
./kmp
```

例如编译 Dijkstra：

```bash
g++ -std=c++20 -O2 "06-graph/6.5.Dijkstra.cpp" -o dijkstra
./dijkstra
```

例如编译快速排序：

```bash
g++ -std=c++20 -O2 "08-sort/8.5.快速排序.cpp" -o quick_sort
./quick_sort
```

---

### 3. 使用 clang++

```bash
clang++ -std=c++20 -O2 "05-tree/5.1.二叉树.cpp" -o tree
./tree
```

## 使用说明

### 文件命名说明

当前文件命名采用：

* 章节号
* 小节号
* 中文主题名

例如：

* `1.2.单链表头结点.cpp`
* `4.2.kmp算法.cpp`
* `6.5.Dijkstra.cpp`
* `8.8.归并排序.cpp`

这种命名方式有两个优点：

1. 与知识点顺序一致，便于复习与记忆
2. 文件名本身即可表达主题，便于快速定位

注意：由于文件名中包含中文，命令行编译时建议始终使用引号包裹路径。

### 当前组织方式说明

本项目当前更偏向于：

* **学习型仓库**
* **复习型仓库**
* **模板积累型仓库**

因此暂未强制统一为：

* 单一可执行程序
* 多文件链接工程
* CMake 完整构建项目

这是一种有意保留的设计，目的是降低学习和复习成本，使每个知识点都可以独立理解、独立运行、独立修改。

## 建议使用方式

建议你使用本仓库时遵循以下步骤：

1. 先学习理论定义与基本操作
2. 再阅读对应 `.cpp` 文件实现
3. 自己独立重写一遍
4. 给关键结构补充测试数据
5. 总结时间复杂度、空间复杂度和适用场景
6. 能够脱离代码口述核心思想

对于准备复试 / 机试的同学，建议重点关注：

* 链表基本操作
* 栈与队列应用
* KMP
* 二叉树遍历
* 图的遍历与最短路径
* 并查集
* 各类排序算法
* 查找树与哈希表
